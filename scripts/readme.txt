The files is this folder are for automating the launch of the MCS. 
The main files are 'script_simulation' and 'script_field' for launching a simulation and field test, respectively. 

The systen is launched by running the following command in the terminal "./[name of file] "

If permission is denied access to all files in this folder must be added manually. 
This is done by running the following command in the terminal for each file "chmod +x [filename]"
